# For DRiP Collectors

## Verifying Ownership : 

### Linking Wallet Address: 

**Command**: `/register` wallet_address=YOUR_WALLET_ADDRESS

**Example**: `/register` wallet=5dC...aN9

**Description**: Collectors must run this command to link their Solana wallet address to their Discord account. DRiPBot uses this address to verify NFT ownership.

## Accessing Exclusive Content
Once verified, collectors will gain automatic access to designated channels, enabling them to join exclusive discussions, events, and announcements curated for NFT holders.
